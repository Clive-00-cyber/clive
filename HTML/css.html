CSS Programming

The styles language for web pages and use to add beutifullness to the page

About CSS

CSS (Cascading Style Sheets) is a stylesheet language ussed to describe the presentation and visual layout.

Key Features

  • Easy to learn and use
  • Responsive design
  • Animation and transitions
  • Cascading and specification
  • Box modelling
  • Selectors

Example Code

/*CSS style example*/
                    body {
                    line-height: 1.6;
                    color: #333;
                    background-color: #f5f5f5;
                }
                
                .container {
                    width: 90%;
                    max-width: 1200px;
                    margin: 0 auto;
                    padding: 0 20px;
                }

Popular Uses

CSS is used for:

  • Styling text and Fonts
  • Layout and positioning
  • Color and background
  • Animation and transitions
  • Spacing and alignment
  • Responsive design
