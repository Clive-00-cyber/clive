GÉNÉRATEUR DE TABLEAUX DE NOMS - VERSION 2.0
=============================================

DESCRIPTION
-----------
Application permettant de créer des tableaux aléatoires de noms sans répétition sur les mêmes lignes.

FONCTIONNALITÉS
---------------
- Ajout et gestion de listes de noms
- Génération de tableaux 8x4 aléatoires
- Export PDF des tableaux
- Sauvegarde/chargement des listes
- Interface intuitive en français

INSTALLATION SUR UN AUTRE ORDINATEUR
------------------------------------

MÉTHODE 1 (Recommandée) - Installation automatique :
1. Exécutez "Installateur_Generateur_Tableaux_Noms.exe"
2. Suivez les instructions de l'installateur
3. Un raccourci sera créé sur le Bureau et dans le Menu Démarrer

MÉTHODE 2 - Version portable :
1. Copiez le fichier "GenerateurTableauxNoms.exe" sur l'ordinateur cible
2. Double-cliquez pour lancer l'application
3. Aucune installation nécessaire

CONFIGURATION SYSTÈME REQUISE
-----------------------------
- Windows 7, 8, 10 ou 11
- 100 MB d'espace disque libre
- Mémoire RAM: 512 MB minimum

UTILISATION
-----------
1. Ajoutez des noms (minimum 8 requis)
2. Cliquez sur "Générer le tableau"
3. Exportez en PDF si nécessaire

SUPPORT
-------
En cas de problème, vérifiez que:
- Windows est à jour
- Les droits administrateur sont accordés si nécessaire
- L'antivirus ne bloque pas l'application
